# 資料夾說明

這個資料夾用於存放訓練好的模型檔案。

## 檔案說明

執行訓練後，會產生以下檔案：

- `bird_classifier.keras` - Keras 3.0 格式的模型（主要使用）
- `bird_classifier.h5` - HDF5 格式的模型（舊版相容）
- `training_history.png` - 訓練過程的視覺化圖表

## 模型資訊

- **架構**: MobileNetV2 (遷移式學習)
- **輸入大小**: 224 x 224 x 3
- **輸出**: 3 個類別的機率分佈
- **類別**: 白尾八哥、家八哥、林八哥

## 注意事項

⚠️ 模型檔案通常較大（約 10-50 MB），在上傳到 GitHub 時請注意：

1. 如果檔案 < 100MB，可以直接提交
2. 如果檔案 > 100MB，需要使用 Git LFS
3. 或在應用中實作動態下載邏輯

詳細說明請參考專案根目錄的 `DEPLOYMENT.md` 文件。
